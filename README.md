# 🚲 Smart Bike Rental Mobile App (SMBK)

A modern, feature-rich Flutter application designed to revolutionize urban mobility through seamless smart bike rentals. 

SMBK (Smart Bike) provides users with an intuitive and lightning-fast experience to locate, unlock, and ride bicycles across the city. Built with cutting-edge mobile technologies, it ensures high performance, real-time tracking, and a premium user interface.

## ✨ Key Features

*   **📍 Real-time Location Tracking**: Interactive maps powered by `flutter_map` and `geolocator` to instantly find nearby available bikes.
*   **📷 Instant QR Unlocking**: Effortless bike unlocking using an integrated, lightning-fast QR code scanner (`mobile_scanner`).
*   **🎨 Premium UI/UX**: A sleek, modern interface utilizing custom typography (`google_fonts`) and smooth SVG rendering for crisp visuals.
*   **⚡ Robust State Management**: Powered by `flutter_riverpod` for a responsive, glitch-free, and scalable application architecture.
*   **🔒 Secure & Fast Auth**: Built-in HTTP client integration for fast, secure remote authentication and user management.

## 🛠 Tech Stack

*   **Framework**: [Flutter](https://flutter.dev/)
*   **State Management**: Riverpod
*   **Maps & Geolocation**: flutter_map, geolocator, latlong2
*   **Camera/QR**: mobile_scanner
*   **Design/Assets**: Google Fonts, flutter_svg
